# To-do-week-list

Недельный список задач для тренировки тайм-менеджмента.

Интерфейс построен на **React** и **[Ant Design](https://ant.design/)**.

## Возможности

- Создание задач с названием, описанием и днём недели
- Отображение задач в 7 колонках (понедельник — воскресенье)
- Перетаскивание задач между днями (drag & drop)
- Удаление задач с подтверждением
- Сохранение данных в `localStorage`
- Фильтрация по периоду даты создания

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```
